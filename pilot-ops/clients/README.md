# Clients

Each client pilot gets its own folder here.

## Naming
Use lowercase kebab-case slugs.

Examples:
- `northshore-web-studio`
- `verdant-legal`
- `oak-ember-home`

## Suggested contents per client
These come from the base pilot template:
- `CLIENT.md`
- `CHANNELS.md`
- `WORKFLOWS.md`
- `TONE.md`
- `BOUNDARIES.md`
- `SUCCESS.md`
- `SAMPLES.md`
- `FAQ.md`
- `LOG.md`
