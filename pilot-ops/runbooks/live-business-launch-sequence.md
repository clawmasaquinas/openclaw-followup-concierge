# Live Business Launch Sequence

This is the practical order of operations for moving from local buildout to real business activity.

## Already done locally
- offer packaged
- outreach materials written
- discovery/demo materials written
- pilot proposal + intake + kickoff flow created
- local landing site scaffold built
- pilot workspace generator built

## Next local actions Ted can do immediately
1. review the landing site copy and positioning
2. choose the first buyer segment to test
3. populate 10 real prospects in the outreach tracker
4. send the first small outreach batch
5. use the pilot proposal + intake flow when someone bites

## External/public actions still not taken
- no public site deployed
- no domain connected
- no real outreach sent yet
- no live pilot client onboarded yet

## Once Ted approves external go-live work
The next sequence should be:
1. deploy static landing page
2. connect response/contact path
3. send first outreach batch
4. book discovery calls
5. onboard first pilot client into `live-business/pilot-ops/clients/`

## Rule
Do not confuse readiness with launch.
The local system is now materially more real than before, but revenue still requires real prospects and real conversations.
