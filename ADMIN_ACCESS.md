# Admin Access

## Hosted admin UI
Use:
- `https://live-business.vercel.app/api/review-ui`

## How to access
Enter the admin token stored locally at:
- `live-business/.local-secrets/admin-token.txt`

## What the admin UI currently shows
- lead name
- business
- email
- primary channel
- problem summary
- desired 14-day win
- submitted timestamp
- Discord notification status

## Notes
This is a minimal protected admin view, not a polished CRM. It is meant for operational visibility in the first live version.
